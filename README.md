## LineBot_Translate(AWS)

### Requirepment 
- LINE Development
- AWS
    - Lambda
    - API Gateway
    - Translate
    - Cloud9
- Ruby 2.5

### How to create
- Build enviroment in Cloud9
- write code
- create instances of Lambda and Api Gateway
- publish webhook in Line Development
- Send a message to LineBot
